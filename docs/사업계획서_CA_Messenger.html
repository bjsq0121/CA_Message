1. 프로젝트 개요

1.1 배경 및 필요성

현재 사내 커뮤니케이션은 다우오피스 메신저를 사용하고 있으나, 다음과 같은 한계가 존재합니다:

현재 (다우메신저)

  • ERP(CA_NERP) 사용자 체계와 별도 관리
  • 인당 월 4,000~5,000원 지속 비용
  • 커스터마이징 불가 (벤더 의존)
  • ERP 업무 데이터 연동 한계
  • UI/기능 변경 요청 시 벤더 대응 필요
  • 데이터가 외부 클라우드에 저장

도입 후 (CA Messenger)

  • ERP 사용자/조직 자동 동기화
  • 추가 라이선스 비용 ₩0
  • 완전한 소스 보유, 자유 커스터마이징
  • ERP 주문/생산/공시 알림 연동 가능
  • 자체 기능 추가/변경 즉시 가능
  • 데이터 사내 IDC 저장 (데이터 주권)

1.2 프로젝트 목표

  • 다우메신저를 대체하는 사내 전용 메신저 자체 구축
  • 웹, 데스크톱(Windows/Mac), 모바일(iOS/Android) 멀티 플랫폼 지원
  • CA_NERP ERP와 MSA(마이크로서비스) 기반 독립 운영
  • 부서/조직 기반 채팅방 자동 관리로 업무 효율 극대화

5. 유연성과 확장성

주문 접수 알림불가채팅방에 자동 전송 가능

5.2 확장 가능 시나리오

1단계 — 즉시
기본 메신저
1:1/그룹 채팅, 파일 전송, 읽음 확인
2단계 — 3개월
ERP 연동
주문/생산/결재 알림, 부서 자동 채팅방
3단계 — 6개월
AI 비서
AI 챗봇 (재고 조회, 단가 문의, FAQ)
핵심 확장성: 자체 소스 보유 + MSA 구조로, 메신저를 단순 채팅 도구가 아닌 ERP 통합 커뮤니케이션 허브로 발전시킬 수 있습니다. 다우메신저로는 불가능한 비즈니스 로직 연동이 가능합니다.
